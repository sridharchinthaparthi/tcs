# TCS NQT Practice Platform

A comprehensive front-end web application designed to help students prepare for the TCS NQT (National Qualifier Test) 2026. The platform provides a rich repository of practice questions covering aptitude, reasoning, verbal ability, programming, and coding challenges.

## 🚀 Features

- **1000+ Practice Questions**: Extensive question bank specifically tailored for TCS placement exams.
- **Five Core Modules**:
  - **TCS Aptitude**: Numerical ability, percentages, probability, and pure math concepts.
  - **TCS Logical Reasoning**: Pattern recognition, data interpretation, and deductions.
  - **TCS Verbal Ability**: Reading comprehension, vocabulary, and grammar.
  - **TCS Programming MCQs**: C/C++, Java, Python fundamentals, and pseudocode.
  - **TCS Coding Problems**: Hands-on coding challenges with optimal solutions.
- **Theme Toggling**: Built-in Dark and Light mode support for comfortable usage.
- **Responsive Design**: Fully mobile-responsive layout for practicing on any device.
- **SEO Optimized**: Fully optimized for search engines with meta tags, OpenGraph data, and JSON-LD structured data.

## 📂 Project Structure

- `index.html`: The main entry point, containing the structural layout, views, and SEO metadata.
- `style.css`: Core stylesheet featuring modern CSS variables, responsive grids, and animations.
- `app.js`: Application logic handling view switching, dark mode, and dynamic content rendering.
- `encrypt_data.py`: Utility Python script for processing and encrypting question data.
- `data/`: Directory storing the application's practice questions data.
- `sitemap.xml` & `robots.txt`: SEO configuration files.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3, Vanilla JavaScript (ES6+).
- **Typography**: Inter and Outfit (via Google Fonts).
- **Icons**: Inline SVG graphics for optimized performance.

## 💻 Getting Started

This is a static web application and requires no complex build tools or backend servers to run locally.

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd tcs
   ```

2. Open `index.html` directly in any modern web browser.
   ```bash
   # Or using a simple local server, for example:
   python -m http.server 8000
   ```
